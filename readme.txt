To run:
	python3 ftp-server.py

	python3 ftp-client.py

	The client will ask for a ip address and port number. 
	Once connected the following commands are possible:
		store
			Follow prompt to enter file name
			Store a file on the server
		retrieve
			Folow prompt to enter file name
			Get a file from the server
		quit
			Stop sever connection
		list
			List all files on server
